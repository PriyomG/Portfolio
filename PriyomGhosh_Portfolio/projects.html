ISP Grade 12 - Pathfinder Adventure

Grade 12 ISP project made by Priyom Ghosh and Henry Zhang. Pathfinder Adventure is a survival-based game where the user tries to navigate their way through several environments using the materials given to them. Their are 3 levels at the moment, each with their own task. Level one is a tutorial level in which the user learns the basic mechanics of the game and tries to find their way to their first class of the school day. The second level consists the user trying to navigate the map to complete yet another task. In the third and final level, the user has to complete a timed challenge in which there are multiple tasks involving the use of navigation through an environment.
